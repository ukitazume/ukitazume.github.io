Apr 23, 2017   #raspberrypi 

Raspberry Pi 3を秋葉原で買ってきた。4400円くらい。 それにカメラ、LCDケーブルとか買ってたら、15000円くらい費やした。

Raspberrypi設定

  • ディクスを焼く
  • 有線Lanでつなぐ
  • bootディクレトリにsshフォルダを作る
  • arp -a でip検索
  • ssh pi@ip

Wifiの設定

  • https://www.raspberrypi.org/documentation/configuration/wireless/wireless-cli.md

カメラ

  • sudo raspi-configでenableに
  • http://ami-gs.hatenablog.com/entry/2014/03/11/224452

楽しい。マイコンの回路の方GPIOなどを学んで、ブレッドボード等を買ってくる必要がある。
